## 🛠️ Building Your Own Skill

```typescript
// skills/my-custom-skill/index.ts
import { defineSkill, z } from "@openclaw/sdk";

export default defineSkill({
  name: "my-custom-skill",
  description: "Does something amazing",
  inputSchema: z.object({
    query: z.string().describe("What to search for"),
  }),
  async execute({ query }) {
    // Your logic here
    return { result: `Found: ${query}` };
  },
});
```

```bash
openclaw-power-skills publish my-custom-skill
```
